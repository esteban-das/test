Mode de jeu

Jouez au version modoerne de mode populaire et decouvrez de nouvelle experience innovantes de Arsenal

death match :
ut est de tuer autant d'adversaires que possible dans un temps limité ou jusqu'à ce qu'un nombre de frags (éliminations) soit atteint. Les joueurs s'affrontent dans une arène ou une carte de jeu

conquest :
Le Conquest est un mode de jeu dans les jeux vidéo multijoueurs où deux équipes s'affrontent pour le contrôle d'un certain nombre de points sur une carte de jeu. Les joueurs doivent capturer et tenir ces points stratégiques pour gagner des points de contrôle. Plus l'équipe contrôle de points, plus elle gagne de points de contrôle. L'objectif est de gagner le plus de points possible avant la fin de la partie.

training :
Le Training, également appelé mode entraînement ou mode pratique, est une fonctionnalité dans les jeux vidéo qui permet aux joueurs de s'entraîner et de perfectionner leurs compétences sans avoir à jouer contre d'autres joueurs ou des adversaires contrôlés par l'ordinateur
